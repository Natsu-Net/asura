API Documentation

Get all the manga

Parameters:

  • page: number
  • limit: number

Example: /api/?page=1&limit=10

Get a manga

GET /api/:slug

Parameters:

  • slug: string

Example: /api/solo-leveling

Get pages of a chapter

GET /api/:slug/chapter/:chapter

Parameters:

  • slug: string
  • chapter: number or id

Qeury param

  • includeChapters: boolean

Example: /api/solo-leveling/chapter/1
